# وعي · Waey

منصة **الذكاء السلوكي المالي** لطلاب الجامعات السعوديين — تجربة مثل Duolingo لكن للمال.
تطبيق React (Vite) ثنائي اللغة (عربي/إنجليزي) مع ثيم داكن/فاتح، متجاوب لكل الأجهزة، ورمز الريال السعودي الرسمي.

## المزايا
- صفحات تسويق: هبوط (Landing) · عن وعي (About) · تسجيل دخول (Login).
- تطبيق كامل: الرئيسية، التحليلات، شات وعي (ذكاء اصطناعي)، الاستثمار، المزيد.
- المحاور الأربعة: محرك الشخصية الإنفاقية · المقارنة المجهولة · التحديات الأسبوعية · لوحة الالتزام (بأسلوب Whoop).
- شات وعي يرسم رسومات بيانية (توزيع/توقّع أسبوعي-شهري-سنوي) ويعطي عرض قرض من راتب الشهر الجاي.

## التشغيل محلياً
يلزمك Node.js 18+.

```bash
npm install
npm run dev
```
افتح الرابط اللي يطلع (عادة http://localhost:5173).

## البناء للنشر
```bash
npm run build      # ينتج مجلد dist/
npm run preview    # معاينة نسخة الإنتاج
```

## ملاحظة عن الذكاء الاصطناعي
شات «وعي» يستدعي واجهة Anthropic مباشرة (تشتغل داخل بيئة Claude بدون مفتاح). في موقع منشور بشكل مستقل، هذا الاستدعاء يحتاج **خادم وسيط (proxy) يحمل مفتاح API**. بدونه، يرجع الشات تلقائياً إلى **الردود الذكية المحلية + الرسوم البيانية** (تشتغل بدون إنترنت). لتفعيل الذكاء الكامل في الإنتاج، أنشئ نقطة وسيطة تمرّر الطلب إلى `https://api.anthropic.com/v1/messages` مع مفتاحك، وبدّل عنوان `fetch` في `src/App.jsx` إليها.
