# Issue Tracker

This repo tracks work in GitHub Issues for `Aiwadod/waey`.

## Tooling

- Prefer the `gh` CLI for issue and PR operations when credentials are available.
- Keep issue titles concise and task-oriented.
- Link implementation PRs to the related issue when one exists.

## Defaults

- Repository: `https://github.com/Aiwadod/waey`
- Issue tracker: GitHub Issues
- Pull requests: GitHub Pull Requests

## Fallback

If GitHub access is unavailable, record temporary notes under `.scratch/` and migrate them to GitHub Issues when access returns.
