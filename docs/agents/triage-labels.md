# Triage Labels

Use this default five-label vocabulary for issue triage.

| Role | Label |
| --- | --- |
| Maintainer needs to evaluate | `needs-triage` |
| Waiting on reporter or requester | `needs-info` |
| Fully specified and ready for an agent | `ready-for-agent` |
| Needs human implementation or judgment | `ready-for-human` |
| Will not be actioned | `wontfix` |

Do not create duplicate labels with near-identical meanings. If the GitHub repository later standardizes different label names, update this file and `AGENTS.md`.
